## Building

### Native
- [Install Rust](https://www.rust-lang.org/learn/get-started)
- Clone the repo
- Run `cargo run --release`

### Web
- [Install Rust](https://www.rust-lang.org/learn/get-started)
- Install trunk: `cargo install trunk --locked`
- Clone the repo
- Run `trunk build --release`
- The dist folder will contain the web files to host
